# SOC Analyst Lab

This repository documents a realistic end-to-end SOC analyst lab built to demonstrate
Tier 1 alert triage, Tier 2 investigation, and basic incident response in a hybrid on-prem/Azure environment.

## Objectives
- Practice SOC workflows used in real environments
- Gain hands-on experience with Microsoft Sentinel and KQL
- Build a professional portfolio for junior SOC analyst roles

## Technologies
- Hyper-V
- Windows Server (Active Directory)
- Windows 10
- Linux
- Microsoft Sentinel
- KQL
